Vue Composition API

Back in June VueJS released an RFC (request for comments) called Function-based Component API. There were some that got really upset with this proposal and it caused a lot of conversation in the community. This week the Vue Core Team released an updated version of the RFC titled Composition API.

I think a lot of the complaints came from everyone thinking that this is how they would have to write their Single File Components (SFC) and that just wasn't the case. There are some important changes to the RFC and if you want to learn more about the Composition API there is an entire documentation site dedicated to it.

https://vue-composition-api-rfc.netlify.com/#summary
